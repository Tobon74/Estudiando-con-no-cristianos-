# Estudiando con No-Cristianos

Proyecto SPA (single-page) para acompañar estudios bíblicos basados en el material "Studying with non-Christians" (Dr. Douglas Jacoby). Contiene preguntas, guías y recursos pensados para estudios personales y en grupo.

## Descripción

Este repositorio contiene una versión web ligera (Index.html) con secciones de estudio organizadas en tarjetas desplegables. Está pensado para uso offline/online como apoyo para discipulado y evangelización.

## Inicio rápido

1. Clona el repositorio:

   git clone https://github.com/Tobon74/Estudiando-con-no-cristianos-.git

2. Abrir `Index.html` en un navegador moderno.

   O sirve la carpeta con un servidor estático (recomendado):

   - Usando Node.js: `npx http-server` (instala http-server si es necesario)
   - Usando Python 3: `python -m http.server 8000`

3. Navega a `http://localhost:8000` (si usas un servidor) o abre el archivo local.

## Estructura del repositorio

- `Index.html` — Interfaz principal (SPA, HTML + CSS + JS embebido).
- `assets/` — (opcional) imágenes, fuentes y recursos estáticos.
- `README.md` — Este archivo.

> Nota: Actualmente el CSS y JS están embebidos en `Index.html`. Puedes separarlos en `css/` y `js/` si prefieres modularidad.
